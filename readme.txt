== Description ==

Simple User Login and Registration system to your Wordpress 

- User can Login for the site use this plugin through username and password.
- New user can also Register for login through username, e-mail and password.
- This plunin also have option to Logout.

== Installation ==

- Instal the latest version of User Login Register Plugin.
- Active the plugin through the 'Plugin' menu in WordPress.
- In Admin Menu, you can find 'User Login Registration'.
- In 'User Login Registration', have admin settings page in where you can find the Shortcode.

== Shortcode ==

[UserLoginRegistration]
